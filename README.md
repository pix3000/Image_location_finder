# Image_location_finder

>**이미지 정보 중 위치(GPS, 경도 및 위도) 정보를 읽기** <br/>
>**경도와 위도를 사람이 알아보기 쉬운 정보로 전환** <br/>
>**txt 파일로 결과값 저장**
